Comandos básico de GitHub

  1. git init: Inicia um novo repositório.
  2. git add .: Adiciona todos os arquivos.
  3. git add index.txt: Adiciona apenas um arquivo.
  4. git commit -m "": permite adicionar uma observação do arquivo a ser salvo no git.
  5. git status: Verifica o status do arquivo.
  6. git log: Mostra o histório do arquivos que houve commit.
